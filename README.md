# ServiceHub - MERN Project

A full-stack MERN web application that connects users with local service providers.

## Features
- User Authentication
- Service Booking
- Admin Panel
- Cloudinary Upload

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Setup

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm start
